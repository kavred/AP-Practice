# AP History Study Tool 🏛️

A modern, web-based flashcard study tool designed to help you quickly memorize the "Must Know" dates and events for AP European History!

## Features ✨
- **Four Historical Periods**: Covers major dates across Period 1 to Period 4 (1450 - present). 
- **Time & Strategy Metrics**: Automatically tracks your completion time and accuracy percentage dynamically as you respond.
- **Dynamic Hints**: Advanced logical hints which reveal the century and decades step-by-step allowing you to jog your memory before taking a hint deduction.
- **Micro-Animations**: Error shake-animations and completion success screens built into a clean, smooth interface.

## Quick Start 🚀

To use the study tool, visit the published GitHub pages site:
👉 [**Access the Live Tool Here**](https://kavred.github.io/AP-Practice/)

Alternatively, you can clone the repository and run it locally:
```bash
git clone https://github.com/kavred/AP-Practice.git
cd AP-Practice
open index.html
```
